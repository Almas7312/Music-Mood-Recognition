# Music-Mood-Recognition


## Overview
Music Mood Recognition is a deep learning project that classifies songs into emotional categories such as Happy, Sad, Romantic, Dramatic, and Aggressive using spectrogram images generated from audio signals.

## Features
- Audio preprocessing and spectrogram generation
- CNN-based mood classification
- Custom labeled dataset
- Deep learning model training and evaluation
- Mood-based music recommendation

## Technologies Used
- Python
- TensorFlow
- OpenCV
- Librosa
- NumPy
- Pandas
- Matplotlib

## Project Workflow
1. Convert audio files into spectrograms
2. Preprocess and label the dataset
3. Train CNN model
4. Evaluate performance
5. Predict mood from unseen songs

## Future Work
- Transformer-based audio classification
- Real-time mood prediction
- Improved recommendation system
